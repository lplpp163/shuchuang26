# 越南語母語審閱包

這個目錄提供一套可追溯、可離線交付的 119 檔越南語母語審閱工具：

- `傳家話_越南語119句母語審閱工具.html`：瀏覽器審閱頁。
- `傳家話_越南語119句語境目錄.json`：中文意圖、劇集／教案位置、說話者與聽者資料來源。
- `傳家話_越南語119句母語審閱可攜包.zip`：HTML、119 個 MP3、manifest、語境目錄、README 與 `SHA256SUMS.txt` 的完整離線包。

目前交付仍是空白工具，不是母語者審閱成果。工具存在、119 檔可播放或自動化測試通過，都不能冒充真人已完成審閱。

## 語境如何追溯

119 個音檔在目前 App 中共有 204 個實際使用位置：

- 91 個劇場音檔對應 175 個位置。資料來自 `lib/models/conversation_episode.dart`，能追到劇集、幕次、提示／選項、中文意圖，以及外婆或孩子的語意說話者與聽者。
- 28 個課程音檔對應 29 個位置。資料來自 `lib/services/app_store.dart`，能追到故事、教學階段與中文意圖；現有結構沒有可靠記錄語意說話者、聽者或親屬身分，因此工具明示「未知」，不從檔名或示範聲線臆測。

同一音檔若在多個位置使用，頁面會列出每一個位置。課程資料的稱謂題只評估句內明示的稱謂／禮貌線索，不能把未知角色補成外婆或其他家人。

## 判定規則

每項「文字自然」「稱謂／禮貌」「音訊清楚」皆有未填、是、否三種狀態；未填不算完成。頁面只在下列條件成立時開放判定按鈕：

- 「可保留」：音檔觸發真實 `ended` 事件、三項全選「是」，並填 1–5 自然度。
- 「需要修訂」：音檔觸發真實 `ended` 事件、三項都已回答且至少一項選「否」、填 1–5 自然度，並在「建議修訂」或「理由／地區差異」至少填一項。

只按播放、載入成功、跳到尾端或只看文字，都不算完整聽完。已判定後若把必填欄位改成不再符合規則，判定會自動清除。

審閱者另須填匿名代碼、日期、主要家庭用語、兒童互動經驗，勾選本人為越南語母語成人及匿名使用同意。請勿填姓名、電話、地址、電子郵件、學校或社群帳號。

## 匯出證據

JSON 是原始紀錄，schema 為 `our-family-says/native-review/v2`。它綁定音訊 `manifestSha256`、`contextCatalogSha256` 與兩份 Dart 來源雜湊，並逐檔保存三項是／否、自然度、修訂、理由、判定、實際播放次數／時間／秒數及完整語境。`completion.complete=true` 只代表 119 檔在形式上都符合必填規則，仍需由團隊核對審閱者資格及內容品質。

CSV 僅供彙整；工具會防止試算表公式注入，但不能取代含雜湊與語境的 JSON 原始紀錄。音檔、文字或使用情境修改後，應重建 manifest／語境目錄，並由母語者重新複聽受影響檔案。

## 產生與重現

在 `正式版/flutter_app` 執行預設生成：

```powershell
python tool/generate_native_review_packet.py
```

完整指定輸入、輸出及音訊前綴：

```powershell
python tool/generate_native_review_packet.py `
  --manifest assets/audio/piper_generation_manifest.json `
  --output "../../交付成果/語言審閱/傳家話_越南語119句母語審閱工具.html" `
  --context-output "../../交付成果/語言審閱/傳家話_越南語119句語境目錄.json" `
  --zip-output "../../交付成果/語言審閱/傳家話_越南語119句母語審閱可攜包.zip" `
  --audio-prefix auto
```

生成器會逐檔核對 manifest 的 bytes 與 SHA-256、確認 119 MP3 集合完全一致，再透過 Flutter 執行期讀取目前劇場與課程資料。沒有 Flutter 時可用 `--context-catalog <已驗證目錄.json>` 重建；生成器仍會驗證目錄本身雜湊、manifest 綁定及目前 Dart 來源雜湊，拒絕漂移資料。相同輸入會產生相同 HTML、語境目錄與 ZIP 位元組。

## 使用可攜包

把 ZIP 完整解壓縮後開啟 `index.html`；不要在壓縮檔預覽視窗內直接開。所有音檔都在 `assets/audio`，不需要網路。若瀏覽器的 `file://` 政策限制播放，可在解壓縮目錄啟動本機靜態伺服器：

```powershell
python -m http.server 18765 --directory "<解壓縮目錄>"
```

再開啟 `http://127.0.0.1:18765/`。可用 `SHA256SUMS.txt` 核對包內 HTML、manifest、語境目錄與 119 個 MP3 是否未被更動。
