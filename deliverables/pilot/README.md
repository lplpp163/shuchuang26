# 傳家話｜教師家庭延伸情境包建置說明

此目錄只維護一份正文來源：

- `傳家話_教師家庭延伸情境包_四週試辦版.md`

正式PDF由本機腳本重建，不下載官方素材、不嵌入教育部或其他單位Logo：

```powershell
python .\build_pilot_pack.py
```

只重跑既有PDF的QA：

```powershell
python .\build_pilot_pack.py --qa-only
```

輸出：

- `傳家話_教師家庭延伸情境包_四週試辦版.pdf`：匿名A4正式試辦包。
- `傳家話_匿名試辦彙整台.html`：完全離線的匿名 cohort 計數彙整工具；直接以瀏覽器開啟即可，不上傳、不保存。
- `QA/檢查摘要.json`：頁數、A4尺寸、最小渲染字級、中繼資料、圖片／Logo、必要章節及SHA-256檢查。
- `QA/版面總覽.png`：供人工檢查裁切、近空白頁與表格配置。

## 重建環境

- Windows中文字型：`C:\Windows\Fonts\msjh.ttc`、`msjhbd.ttc`
- Python套件：`reportlab`、`PyMuPDF`、`pypdf`、`Pillow`

## 文件界線

- 本包是可執行但尚未取得真人試辦結果的工具，不是官方教材、認證教案、合作方案或採購文件。
- 唯一銷售單位是 `8,000元／一班／一次四週／最多30戶` 的待訪談假設，不是公開牌價、報價或成交證據，更不是長期方案價格。
- 故事種子由孩子選第一棒、經成人 PIN 確認家人說法後形成「四關生活任務」；孩子完成第三棒後「家庭故事護照」才蓋章。成人另可自願複製不含家庭內容或各筆識別的匿名試點摘要。不能確認的按鈕逐字均在正文標為「發布前對照項」。
- 匿名彙整台只接受 `hometongue-pilot-summary-v1` 固定白名單；額外欄位、疑似個資、未知題材、重複 cohort 或不一致漏斗會整批拒絕。它只加總開始／成人完成／孩子完成與明確分母漏斗率，不合併缺少樣本數權重的平均時間。
- 工具內建的合成示例會永久標示為非證據；匿名自陳計數也不等於真人觀察、學習成效、母語正確性、代表性抽樣或外部查核。
- 官方入口查核日為2026-07-14；超過14天或正式帶領前須重新查核。
- 若正式產品尚未出現正文採用的最終用語，先修正產品或正文再重建，不得聲稱已完全逐字一致。
