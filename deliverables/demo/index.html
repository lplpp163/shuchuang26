      正式版 AI 架構

      真正執行的不是一頁 HTML

      畫面只是入口；文件會送到 FastAPI 後端，交由多模態模型同時理解影像與文字，再由結構化 schema 與本地風險規則檢查結果。

      1. 1
        輸入驗證

        檔案 MIME、大小、文字長度與同意狀態。

      2. 2
        多模態 Responses API

        圖片使用 vision input；PDF 同時讀取抽出的文字與逐頁影像。

      3. 3
        Pydantic Structured Outputs

        文件類型、欄位、證據、信心、警示與行動都必須符合固定格式。

      4. 4
        確定性安全閘門

        扣款、欠薪、解約、證件與人身安全等情境，不讓模型自行下法律結論。

      5. 5
        可重現評測

        使用完全合成的標準案例與對抗案例檢查欄位、風險召回與拒答行為。
