傳家話 Flutter 專案

此目錄是主專案 `正式版/flutter_app/` 的初賽送審鏡像。Flutter Web 是本次主要可操作作品；這裡另收原始碼、設定、測試與說明，不收編譯快取或私人環境檔。

直接操作 Flutter Web 初賽作品開啟 App → 啟動、功能與實測狀態README → 免費套件與專案設定pubspec.yaml → 直接相依套件與授權授權 → 技術驗證紀錄驗證 → 應用入口main.dart → 本地任務引擎local_task_engine.dart → 本地任務引擎測試test →

鏡像結構

source/ ├── lib/ Flutter 應用原始碼 ├── test/ 可重跑的測試 ├── pubspec.yaml 套件與資源 ├── pubspec.lock 實際鎖定版本 ├── OPEN_SOURCE_NOTICES.md ├── analysis_options.yaml └── README.md 操作、邊界與驗證

鏡像由 `scripts/sync-deliverables.ps1` 自動建立。如果本頁任一連結尚未存在，代表最新 Flutter 版本尚未同步，不以舊作品代替。功能是否通過分析與測試，以鏡像 README 和嚴格驗收報告為準。
