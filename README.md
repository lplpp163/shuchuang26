# shuchuang26｜IEC2026 臺灣數創大賞 開會資料

給隊員看的參賽策略與開會資料。

## 怎麼看
- **開會簡報（推薦）**：開 `開會簡報.html`（或 `index.html`，內容相同）。直接用瀏覽器打開即可，裡面有得獎海報截圖、主題候選、分工和議程。
- **文字版**：看 `策略總結.md`。
- 圖片放在 `images/`，請和 HTML 放在同一層，不然海報會破圖。

## 內容
- 為什麼這個比賽商業論述比技術重要（附歷屆得獎海報範例）
- 選組與主題判斷、具體候選
- 初賽前分工（分頭研究、餵 AI 生初稿）
- 餵 AI 的資料範例、關鍵時程、開會議程、團名構思

歷屆得獎作品海報全檔不在此 repo，存在團隊本機的 `歷屆得獎作品/`。
